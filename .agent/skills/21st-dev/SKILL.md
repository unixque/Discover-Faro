# 21st.dev Component Integration Skill

Use this skill when asked to integrate high-quality, professional UI components, blocks, or templates from 21st.dev (the shadcn/ui for AI).

## Workflow
1. **Discovery**: When a new UI component or section is needed, first search or refer to 21st.dev patterns (Hero sections, Navbars, Bento grids, etc.).
2. **Implementation**:
   - Use `npx @21st-dev/cli@latest add <component>` if the specific component identifier is known.
   - Alternatively, fetch the source code from the registry and integrate it manually following the project's design system.
3. **Styling**: Ensure components are consistent with Tailwind CSS and the existing shadcn/ui theme in the project.
4. **Animation**: Leverage `framer-motion` (Motion) for all transitions and interactions to ensure a premium feel.

## Principles
- **Aesthetics First**: Prioritize "Wow" factor and premium design.
- **Responsiveness**: All components must be fully responsive.
- **Accessibility**: Follow WCAG standards for all interactive elements.
